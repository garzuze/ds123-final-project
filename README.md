# Projeto final DS123: lista de tarefas

Esse projeto contém um código em C que permite ao usuário administrar suas tarefas! Com um menu interativo, ele pode adicionar, editar e deletar tarefas. Ao final da execução do programa é gerado um arquivo.txt contendo sua lista de tarefas.

**Conceitos utilizados**
- Estruturas básicas como loops e condicionais;
- Ponteiros;
- Manipulação de memória;
- Manipulação de arquivos;
- Struct (Linked List)  [ainda não vimos, mas resolvi utilizar para treinar estruturas de dados]
